# Steps on How to Create A Github Page

1.) Create a New Repository
2.) Enter the name of the repository as username.github.io with the username part being replaced by your github username.
3.) Toggle Add README as on.
4.) Create the Repository
5.) Go to the Settings of your Repository
6.) In the "Code and Automation" Section, click Pages
7.) Under "Build and Development", under "Source", select Deploy from Branch
8.) Under "Build and Development", under "Branch", select your branch.
9.) Save and you are good to go.
